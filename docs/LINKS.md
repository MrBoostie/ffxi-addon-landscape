# FFXI Addon Research Links

## Documentation
- Windower Docs — official setup, Lua APIs, and addon/plugin guides: <https://docs.windower.net/>
- Windower Support Forum — troubleshooting/help threads and historical fixes: <http://www.ffxiah.com/forum/forum/170/support/>
- Windower Discord — community support and real-time discussion: <https://discord.gg/v6pk6uy>

## Repo Collections (many addons in one place)
- ProjectTako addons — mixed utility pack for Windower/Ashita (React, PartyBuffs, FastCS, etc.): <https://github.com/ProjectTako/ffxi-addons>
- Ivaar Windower addons — automation + QoL set (Auto* job helpers, PorterPacker, SellNPC, Trade tools): <https://github.com/Ivaar/Windower-addons>
- Icydeath addons — large addon bundle/fork collection including GearSwap, HealBot, automation and utility scripts: <https://github.com/Icydeath/ffxi-addons>
- lili-ffxi addons — modern actively-maintained Windower QoL tools (chat fixes, trackers, finder helpers): <https://github.com/lili-ffxi/FFXI-Addons>
- DiscipleOfEris addons — era/private-server flavored Windower variants and utility forks: <https://github.com/DiscipleOfEris/Windower4Addons>

## Focused Addons / Tooling
- superwarp — text-command travel helper for Home Point / Survival Guide / Waypoint style warps: <https://github.com/AkadenTK/superwarp>
- Skillchains — active skillchain window/visualization and timing support in combat: <https://github.com/Ivaar/Skillchains>
- parse — combat parser for damage/performance visibility: <https://github.com/flippant/parse>
- HealBot — automated healing assistant logic and party support behavior: <https://github.com/lorand-ffxi/HealBot>
- autoAssist — auto-assist target helper for coordinated party combat: <https://github.com/ekrividus/autoAssist>

## Related UI Projects
- XIVHotbar — FFXIV-style hotbar UI for FFXI command/ability layout: <https://github.com/Akirane/XIVHotbar>
- XIVHotbar2 — newer fork/iteration of XIVHotbar with extended config/theme behavior: <https://github.com/Technyze/XIVHotbar2>
